CSS Grid

I coded the layout of the book "Grid Systems In Graphic Design" in CSS grid.

Discovering the powerful CSS grid inspired me to create and experiment with content flexible layouts. I was inspired by Swiss Graphic Design and tried to recreate some of their designs in code.

I played with Naef blocks to make them responsive. Change the width of your browser, or check it out on different devices to see what happens. The graphics were originally black and white, and I added color using CSS code. This saved 15% - 30% on image size, which made the loading time faster.
